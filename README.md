# 🚀 Front-End Projects Collection

A collection of front-end projects built by following the <a href="https://roadmap.sh/frontend">Frontend Developer Roadmap</a> from <a href="https://roadmap.sh">roadmap.sh</a>.

These projects focus on improving core front-end development skills including:

- Semantic HTML
- Modern CSS
- Responsive Design
- Accessibility
- UI Components
- Layout Techniques
- Interactive Elements
